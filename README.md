# render-vs

Comparadora visual de código generado por modelos de IA. Renderiza HTML y JSX/React lado a lado con auto-scroll sincronizado para grabar presentaciones limpias.

## Cómo usar

1. Abrí `index.html` en el navegador
2. En cada panel: subí un archivo (HTML/JSX) o pegá el código
3. Personalizá los nombres de cada modelo
4. Clickeá **Comparar**
5. Mové el mouse para ver los controles
6. **Play** → countdown 5 segundos → scroll automático (UI se oculta para grabar)

## Features

- Soporte para HTML puro y JSX/React (con Tailwind, framer-motion, lucide-react)
- Pantalla dividida con divisor arrastrable
- Auto-scroll sincronizado con velocidad configurable
- Countdown antes de arrancar (tiempo para empezar a grabar)
- UI completamente oculta durante el scroll
- Labels siempre visibles
- Descarga de cada panel como HTML
