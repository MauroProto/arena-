# render-vs

Visual side-by-side comparator for code generated by AI models. Renders HTML and JSX/React with synchronized auto-scroll for clean recording.

## How to use

1. Open `index.html` in your browser
2. On each panel: upload a file (HTML/JSX) or paste code
3. Customize the model names
4. Click **Comparar**
5. Move your mouse to reveal controls
6. **Play** → 5-second countdown → auto-scroll starts (UI hides for recording)

## Features

- Pure HTML and JSX/React support (with Tailwind, framer-motion, lucide-react)
- Split screen with draggable divider
- Synchronized auto-scroll with adjustable speed
- Countdown before scrolling starts (time to start recording)
- UI completely hidden during scroll
- Labels always visible
- Download each panel as HTML
